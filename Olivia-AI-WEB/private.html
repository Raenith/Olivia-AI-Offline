Kebijakan Privasi Mutlak

Arsitektur Client-Side Edge AI menjamin tidak ada satu kata pun yang meninggalkan perangkat Anda. Privasi Anda bukan opsi, melainkan sistem bawaan.

🛡️ Komitmen 0MB Data ke Cloud

Olivia AI dibangun menggunakan teknologi modern Transformers.js v3 dan akselerasi WebGPU. Seluruh proses pengolahan bahasa alami, analisis emosi, dan penyimpanan memori dilakukan 100% secara lokal pada browser Anda.

1. Pengumpulan dan Pemrosesan Data

Kami memegang prinsip dasar privasi digital: Kami tidak dapat mengambil data yang tidak pernah kami kumpulkan.

Saat Anda mengetik pesan, berinteraksi dengan Olivia Valen (Companion Mode), atau berkonsultasi dengan AIRA ACS (Support Mode), seluruh teks tersebut diproses secara real-time langsung oleh core GPU/CPU komputer Anda sendiri. Tidak ada server perantara, tidak ada log eksternal, dan tidak ada API pihak ketiga yang terlibat dalam merajut obrolan tersebut.

2. Teknologi Penyimpanan Lokal

Untuk memberikan pengalaman interaksi yang kontekstual dan mengingat preferensi Anda, kami menggunakan sistem penyimpanan bawaan browser sandboxed:

  • Riwayat Percakapan (Multi-turn Memory) LocalStorage
    Riwayat chat disimpan per username dan dioptimalkan secara dinamis menggunakan metode sliding window (4 pasangan obrolan terakhir). Data obrolan ini menetap di browser Anda sendiri.
  • Model Weights Caching Cache Storage / IndexedDB
    Ketika Anda pertama kali membuka Kamar Virtual, browser mengunduh model bahasa Gemma4 E2B IT (ONNX). File model ini disimpan di memori browser Anda agar selanjutnya aplikasi dapat dijalankan 100% Offline.

3. Pemrosesan Logika Cerdas AI

Dua modul utama pada ekosistem Olivia AI beroperasi dengan batasan privasi yang ketat:

• Emotion Detection Engine (Olivia): Menganalisis token input teks secara lokal untuk mendeteksi sentimen negatif/positif, kemudian memodifikasi bobot sistem prompt internal agar merespons secara hangat tanpa mengirim data analisis ke luar komputer.

• Local RAG Engine (AIRA ACS): Pencarian dokumen pengetahuan teknis seputar kendala game dilakukan melalui skrip pencarian teks berbasis JSON lokal (Keyword-Matching) tanpa memanggil database cloud eksternal.

4. Kontrol Penuh Pengguna

Karena seluruh data berada di bawah kendali penuh perangkat Anda, Anda memiliki hak mutlak untuk:

• Menghapus seluruh riwayat percakapan secara langsung kapan pun melalui opsi hapus chat di antarmuka game.

• Mengosongkan penyimpanan aplikasi dengan membersihkan data situs (Clear Site Data / Cache) pada pengaturan browser Anda, yang akan langsung menghapus seluruh sisa memori interaksi AI.

5. Perubahan Kebijakan Privasi

Karena aplikasi ini bersifat *fully client-side* (Edge AI), pembaruan di masa mendatang tidak akan pernah merubah fundamental privasi lokal ini. Jika ada perubahan fungsionalitas, kode sumber kami tetap dapat diaudit secara terbuka guna memastikan keamanan data Anda tetap terjaga.
